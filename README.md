# 🎯 SUPERBET SIGNALS

**Robô de Análise de Jogos e Geração de Sinais de Apostas com Probabilidades**

## 📋 Descrição

Superbet Signals é um sistema de análise de dados de apostas que gera sinais com probabilidades de ganho para jogos de futebol. Utiliza análise estatística e padrões de odds para identificar as melhores oportunidades de apostas.

## ✨ Funcionalidades

- ✅ Análise automática de jogos
- ✅ Geração de sinais com probabilidades
- ✅ Cálculo de confiança
- ✅ Análise Over/Under
- ✅ Previsão de vitória (casa/visitante)
- ✅ Interface CLI simples

## 🚀 Como Usar

### Instalação Local

```bash
# Clonar repositório
git clone https://github.com/lopescaua399-blip/superbet-signals.git
cd superbet-signals

# Criar ambiente virtual
python -m venv venv
source venv/bin/activate  # macOS/Linux
# ou
venv\\Scripts\\activate  # Windows

# Instalar dependências
pip install -r requirements.txt

# Executar
python app.py
```
